## ✨ New Features

- **Undo/Redo** — Full undo/redo support with multi-coordinate history tracking
- **Find & Replace** — Search text with wrap-around support for backward search
- **Clipboard** — Copy, cut, and paste functionality
- **Zoom In/Out** — Zoom feature with reset option and persistent zoom state via config
- **Word Wrap** — Soft-wrapping rendering when text reaches max column
- **Goto Line** — Jump to specific line number
- **View Menu** — New menu with zoom and word wrap options
- **Title Bar** — Show current filename and modified state (`*`) in title bar
- **Edit State Tracking** — Track whether buffer has been modified
- **Recent Files** — Recent files management with linked list
- **Config Module** — Centralized config read/write for persistent settings
- **Auto Release** — GitHub Actions workflow for automated build & release with Inno Setup installer
