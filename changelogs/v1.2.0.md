## ✨ Final Release Features

- **Undo/Redo** — Full history support to undo and redo typing or deletions
- **Find & Replace** — Search text, replace specific words, and wrap-around searching
- **Word Wrap** — Toggleable soft-wrapping for long lines, persistent via config
- **Clipboard** — Native Windows integration for Cut, Copy, and Paste
- **Advanced Selection** — Double-click to select word, triple-click to select line
- **Zoom Control** — Zoom in and out using `Ctrl + Mouse Wheel`, persistent via config
- **Auto-Scroll** — Editor automatically scrolls when dragging mouse to select text outside window bounds
- **Trackpad Support** — Smooth horizontal scrolling via two-finger trackpad swipe
- **Goto Line** — Jump to a specific line number easily
- **Recent Files** — Track and open recently accessed files directly from the menu
- **Dirty State Tracking** — Asterisk (`*`) indicator in title bar for unsaved changes
