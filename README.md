# Archivista

Simple text editor application built with C and Win32 API.

## Requirements

Sebelum setup project, pastikan tools berikut sudah terinstall:

| Tool          | Keterangan               | Link Download                                     |
| ------------- | ------------------------ | ------------------------------------------------- |
| **MinGW-w64** | C Compiler untuk Windows | [winlibs.com](https://winlibs.com/)               |
| **CMake**     | Build system             | [cmake.org/download](https://cmake.org/download/) |
| **Git**       | Version control          | [git-scm.com](https://git-scm.com/)               |

Pastikan `mingw64\bin` dan `cmake\bin` sudah ditambahkan ke **System Environment Variable PATH**.

Contoh path MinGW: `D:\apps\winlibs\mingw64\bin`

Untuk mengecek apakah sudah terinstall dengan benar, jalankan di terminal:

```
gcc --version
cmake --version
```

## Setup

### Visual Studio Code

1. Clone repository:

   ```bash
   git clone https://github.com/WildanMZaki/Archivista.git
   cd Archivista
   ```

2. Buka folder project di VS Code:

   ```bash
   code .
   ```

3. Untuk **debug**, jalankan perintah berikut (Wajib Menggunakan `Command Prompt` atau `Powershell`):

   ```bash
   debug
   ```

4. Untuk **build release** (menghasilkan `.exe` final):

   ```bash
   build-release
   ```
